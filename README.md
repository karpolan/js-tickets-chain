# js-tickets-chain
Test task from some Job Interview

## Task
You got a shuffled bunch of tickets. Chain them up and find the point of departure and destination:

### Input
Lviv-Warshaw
Prague-Vienna
Warshaw-Prague
Kharkiv-Kyiv
Kyiv-Lviv

### Output
Traveling from Kharkiv to Vienna

Kharkiv-Kyiv
Kyiv-Lviv
Lviv-Warshaw
Warshaw-Prague
Prague-Vienna
